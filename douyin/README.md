# 重力翻转者 · 抖音小游戏版 🪐

微信版的抖音移植版本。游戏逻辑与微信版**完全一致**，仅通过 `game.js` 顶部的「平台适配层」把 `wx.*` 切换为抖音的 `tt.*`：

```js
const platform = (typeof tt !== 'undefined') ? tt : (typeof wx !== 'undefined' ? wx : {})
```

所以这份 `game.js` 在抖音 / 微信两端都能直接跑。

## 文件结构
- `game.js` — 全部游戏逻辑（含广告、分享、最高分；平台无关）
- `game.json` — 抖音小游戏配置（竖屏）
- `project.config.json` — 抖音开发者工具项目配置（`appid` 待填）
- `share.png` — 转发卡片配图（500×400，5:4）
- `icon-144.png` — 小游戏图标（144×144）

## 如何运行
1. 下载安装 **抖音开发者工具**（字节跳动小程序/小游戏开发者工具）
2. 新建项目 → 类型选**小游戏** → 目录指向本 `douyin/` 文件夹
3. 在抖音小游戏后台拿到你的 **AppID（tt 开头）** 填进 `project.config.json` 的 `appid`
4. 点击编译，即可在模拟器 / 真机调试预览

## 上架与变现要点（抖音）
- **变现门槛**：抖音可在上架前/无访客门槛即开通广告变现（流量主），比微信的「累计 500 UV」宽松
- **资质**：需一份版权证明，**可用「电子版权认证」代替传统软著**（更快更便宜）；纯广告不内购 → **免版号**
- **结算**：约 15 天账单期，月结两次，直接打款到个人银行卡
- **广告位 ID**：开通变现后在后台创建广告位，把 ID 填进 `game.js` 顶部的 `AD_UNIT`：

```js
const AD_UNIT = {
  rewardedVideo: '抖音激励视频广告位ID',
  banner: '抖音Banner广告位ID',
}
```

## 与微信版的差异（仅平台接口层）
| 能力 | 微信 | 抖音 |
|---|---|---|
| 系统/画布/触摸/存储 | `wx.*` | `tt.*`（适配层自动切换）|
| 激励视频 / Banner | `wx.createRewardedVideoAd` / `createBannerAd` | `tt.*` 同名接口 |
| 分享 | `wx.shareAppMessage`（含朋友圈 menus）| `tt.shareAppMessage`（去掉微信专属 menus）|

> 维护建议：游戏玩法/视觉若有改动，改一处后把 `game.js` 同步到另一端即可（两端文件结构一致，仅平台全局变量不同）。
