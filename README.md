# 重力翻转者 🪐

一个极简的微信小游戏：方块自动奔跑，**点击屏幕翻转重力**，躲开上下两侧的霓虹障碍。单手指、竖屏、上瘾。

> 命名说明：游戏名保持**纯中文**「重力翻转者」，是为了让上架时软著保持「选填」——微信规则里，小游戏名称含英文/人名/「软件」字样会触发软著必填。

## 玩法
- 点击屏幕任意位置 → 翻转重力（方块在地板/天花板之间切换）
- 撞到障碍 → 结束
- 躲过一个障碍 +1 分，速度越来越快

### 两种模式
- **无尽模式**：障碍无尽生成，挑战最高分
- **闯关模式**：8 关，逐关解锁，每关机制各异（浮动障碍 / 重力变化 / 提速 / 上下穿缝），通关进度本地保存

> 跨平台：核心 `game.js` 通过平台适配层（`tt`/`wx`）在微信与抖音两端通用；抖音版见 `douyin/`。

## 如何运行
1. 打开「微信开发者工具」
2. 新建项目 → 选择**小游戏**类型，目录指向本文件夹 `gravity-flip/`
3. AppID 可先用「测试号」（也可在 `project.config.json` 里填你自己的小游戏 AppID）
4. 点击编译，即可在模拟器/真机预览

## 文件结构
- `game.js` — 全部游戏逻辑（物理、障碍、计分、特效、界面）
- `game.json` — 小游戏配置（竖屏）
- `project.config.json` — 开发者工具项目配置

## 广告变现（已接入，待填广告位 ID）
代码已接入两个广告位，**广告位 ID 留空时不创建广告**，测试号/预览照常游玩：

- **激励视频**：结束页「▶ 看广告复活」，每局可复活一次
- **Banner**：标题页 / 结束页底部横幅，游戏中隐藏

上架后，在微信公众平台「**流量主 → 广告位管理**」创建广告位，把 ID 填到 `game.js` 顶部：

```js
const AD_UNIT = {
  rewardedVideo: 'adunit-xxxxxxxxxxxxxxxx', // 激励视频广告位 ID
  banner: 'adunit-xxxxxxxxxxxxxxxx',        // Banner 广告位 ID
}
```

填好后重新上传即可生效。

## 可继续扩展的点子
- 障碍样式多样化（尖刺、移动块、需要穿缝隙）
- 背景音乐 + 音效
- 皮肤系统、排行榜（微信开放数据域）

> 注：纯广告变现、不做内购，可免版号上架。如要加内购（虚拟支付），需企业主体 + 游戏版号。
