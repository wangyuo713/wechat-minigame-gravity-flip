**软件简介**：
《重力翻转者》是一款基于平台小游戏框架开发的单机休闲反应类游戏软件。玩家操控一枚在霓虹太空走廊中持续向前飞行的发光方块，通过点击屏幕翻转重力，使方块在地板与天花板之间切换，躲避上下两侧不断出现的障碍。软件以即时反应与刷新最高分为核心乐趣，具备计分、最高分记录、广告复活、好友分享、特效反馈等功能。

## 二、开发背景与目的

随着移动端"即点即玩"小游戏的普及，轻量化、单手操作、碎片时间可玩的休闲游戏需求旺盛。本软件旨在提供一款上手零门槛、节奏明快、画面简洁炫酷的反应类休闲游戏，满足用户在碎片时间内的娱乐与解压需求，同时验证基于平台小游戏框架的轻量游戏开发与商业化（广告变现）方案。

## 四、软件功能结构

软件主要由以下功能模块构成：

```
重力翻转者
├── 主循环模块        逐帧更新与渲染（requestAnimationFrame）
├── 物理与控制模块    重力翻转、速度、边界、碰撞检测
├── 关卡生成模块      障碍随机生成、难度随时间递增
├── 计分模块          实时计分、本地最高分读取与保存
├── 渲染与特效模块    背景星空、霓虹方块、障碍、拖尾、粒子爆炸、屏震、闪屏
├── 界面状态模块      标题页 / 游戏中 / 结束页 三态切换
├── 广告变现模块      激励视频（看广告复活）、Banner 横幅
└── 社交分享模块      好友分享、转发卡片、最高分记录
```

## 五、主要功能与操作说明

### 5.1 启动与标题页
软件启动后进入标题页，显示游戏名称"重力翻转者"、玩法提示"点击屏幕翻转重力，躲开障碍"及历史最高分。点击屏幕任意位置即开始游戏。

【此处插入标题页运行截图】

### 5.2 游戏进行
- **核心操作**：点击屏幕一次 = 翻转一次重力，方块在地板与天花板之间切换运动方向；
- 方块自动持续前进，玩家需控制翻转时机，使方块从上下障碍之间安全通过；
- 每成功躲过一个障碍计 1 分，画面顶部实时显示当前分数；
- 游戏速度随时间逐渐加快，难度递增；
- 触碰障碍即判定失败，触发爆炸粒子、屏幕震动与闪屏特效。

【此处插入游戏进行中运行截图】

### 5.3 结束页
失败后进入结束页，显示本局得分与历史最高分，并提供：
- **「看广告复活」**：观看一段激励视频广告后，可在本局原位复活继续游戏（每局限一次）；
- **「分享给好友」**：将带有本局分数的转发卡片分享给好友；
- 点击其它位置可立即重新开始新的一局。

【此处插入结束页运行截图】

### 5.4 最高分记录
软件通过平台本地存储记录玩家历史最高分。每当本局得分超过历史最高分，自动更新并永久保存（关闭重启后依然保留），在标题页与结束页展示。

## 六、技术特点

1. **单手极简操作**：全程仅"点击"一个交互动作，零学习成本；
2. **逐帧实时渲染**：基于 Canvas 2D 与 requestAnimationFrame 实现流畅动画；
3. **动态难度**：障碍随机生成、速度随时间递增，保证可玩性与挑战性；
4. **丰富的视觉反馈**：霓虹光晕、运动拖尾、粒子爆炸、屏幕震动与死亡闪屏，提升打击感；
5. **轻量化**：核心逻辑单文件实现，包体小、加载快、即点即玩；
6. **合规商业化**：通过平台官方广告组件（激励视频 / Banner）实现变现，无内购、无虚拟支付。

## 七、数据与隐私说明

软件仅在本地存储玩家"最高分"一项数据，不收集用户姓名、身份证、手机号、位置等任何个人敏感信息，不上传用户数据至第三方，不含聊天、评论等用户生成内容功能。
